# SF2 Banks Folder

Ovaj folder sadrži SoundFont (.sf2) banke za reprodukciju MIDI datoteka.

## Struktura foldera:
```
sf2-banks/
├── README.md
├── general-midi.sf2          # General MIDI banka
├── piano-banks/              # Piano banke
│   ├── grand-piano.sf2
│   ├── electric-piano.sf2
│   └── jazz-piano.sf2
├── orchestral/               # Orkestralne banke
│   ├── strings.sf2
│   ├── brass.sf2
│   └── woodwinds.sf2
├── electronic/               # Elektronske banke
│   ├── synth-leads.sf2
│   ├── synth-pads.sf2
│   └── drums.sf2
└── ethnic/                   # Etničke banke
    ├── world-instruments.sf2
    └── percussion.sf2
```

## Preporučene SF2 banke:
- **General MIDI**: Standardna GM banka za opću upotrebu
- **Piano banke**: Različiti tipovi klavira
- **Orkestralne**: Gudački, limeni, drveni puhači
- **Elektronske**: Sintisajzer zvukovi
- **Etničke**: Svjetski instrumenti

## Kako dodati nove banke:
1. Stavite .sf2 datoteku u odgovarajući podfolder
2. Datoteka će se automatski pojaviti u dropdown meniju
3. Odaberite banku prije puštanja MIDI datoteke

## Napomene:
- Podržani format: .sf2 (SoundFont 2.0)
- Maksimalna veličina: 50MB po datoteci
- Preporučeno: Kvalitetne banke s 16-bit rezolucijom

Voice Editor - Napredno uređivanje zvukova

Voice Editor je moćan alat za napredno kreiranje i prilagođavanje zvukova na Yamaha klavijaturama. Omogućava vam detaljnu kontrolu nad svakim aspektom zvuka.

Pristup Voice Editoru

Kako otvoriti Voice Editor

  1. Odaberite zvuk koji želite urediti (ili kopirati)
  2. Pritisnite FUNCTION → Voice Edit
  3. Alternativno: Pritisnite EDIT ako je dostupan na vašoj klavijaturi

Napomena: Neki preset zvukovi ne mogu se direktno mijenjati - morate ih prvo kopirati u User zone.

Struktura Voice Editora

Osnovni ekrani

Voice Editor obično ima nekoliko ekrana/kategorija:

  • Common - zajednički parametri za cijeli zvuk
  • Element - parametri za svaki element zasebno
  • Layer - postavke za Layer tipove zvukova
  • Split - postavke za Split tipove zvukova
  • Effect - detaljne postavke efekata

Common parametri

Volume (Glasnoća)

Ukupna glasnoća zvuka (0-127).

Savjet: Postavite na 100-120 za optimalnu glasnoću bez distorzije.

Pan (Panorama)

Pozicija zvuka u stereo prostoru:

  • L64 - krajnje lijevo
  • 0 - centar
  • R63 - krajnje desno

Reverb Send

Koliko reverb efekta primjenjuje se na zvuk (0-127).

Viša vrijednost = više reverb-a = veći prostor oko zvuka.

Chorus Send

Koliko chorus efekta primjenjuje se (0-127).

Chorus dodaje dubinu i bogatstvo zvuku, posebno koristan za strings i synth zvukove.

Voice Assign Mode

Kako se zvuk svira kada pritisnete više tipki:

  • Poly - polifoni (više nota istovremeno)
  • Mono - monofoni (jedna nota - za solo instrumente)
  • Legato - glatke prijelaze između nota

Element parametri

Odabir elementa

Većina zvukova ima 1-4 elementa. Odaberite element koji želite urediti:

  1. U Voice Editoru odaberite Element kategoriju
  2. Odaberite Element 1, 2, 3 ili 4
  3. Svaki element može se aktivirati/deaktivirati

Waveform (Oscillator)

Osnovni zvuk elementa. Može biti:

  • AWM2 - uzorkovani zvukovi (realni instrumenti)
  • FM - sintetizirani zvukovi (sintisajzer)
  • AN - analogni modeling (stariji sintisajzeri)

Napomena: Na nekim modelima možete mijenjati waveform, na drugima ne.

Filter parametri

Filter oblikuje karakter zvuka:

  • Cutoff Frequency (0-127) - odsijecanje frekvencija
    • Niska vrijednost = tamniji, mutniji zvuk
    • Visoka vrijednost = svjetliji, oštriji zvuk
  • Resonance (0-127) - naglasak na cutoff frekvenciji
    • Viša vrijednost = zvižduk na cutoff točki
    • Koristi se za efekte poput wah-wah
  • Filter Type - Low Pass, High Pass, Band Pass, itd.

EG (Envelope Generator)

Kontrolira kako se glasnoća mijenja kroz vrijeme:

  • Attack Time (0-127) - koliko brzo zvuk dosegne punu glasnoću
    • 0 = trenutno (udarac, poput bubnja)
    • 127 = vrlo sporo (violinski glaz)
  • Decay Time (0-127) - koliko brzo padne nakon attack-a
  • Sustain Level (0-127) - razina dok držite tipku
    • 100 = puna glasnoća dok držite
    • 0 = zvuk nestane čak i ako držite tipku
  • Release Time (0-127) - koliko brzo se zvuk gasi nakon puštanja tipke
    • 0 = trenutno
    • 127 = vrlo sporo (dugačak tail)

ADSR primjeri

Piano: Kratak Attack (10-20), srednji Decay, visok Sustain, srednji Release

Strings: Srednji Attack (30-50), kratak Decay, visok Sustain, dug Release (80-100)

Percussion: Trenutni Attack (0-5), kratak Decay, nizak Sustain, kratak Release

LFO (Low Frequency Oscillator)

Dodaje oscilacije i pokretanje zvuku:

  • Waveform - oblik oscilacije (sine, triangle, square, sawtooth)
  • Speed/Rate - brzina oscilacije
  • Delay - odgoda prije početka oscilacije
  • Fade In - postupno povećanje oscilacije

Modulation Target - što LFO modulira:

  • Filter Cutoff - wah-wah efekt
  • Pitch - vibrato
  • Amplitude - tremolo

Detune

Fino podešavanje visine tona elementa (-64 do +63 centova).

Koristite za:

  • Usklađivanje elemenata u Layer zvukovima
  • Kreiranje "fat" zvukova (mali detune između elemenata)
  • Kreiranje "chorus" efekta

Velocity Control

Kako element reagira na snagu pritiska (velocity):

  • Velocity Sensitivity - osjetljivost na pritisak
  • Velocity Range - raspon brzine pri kojem se element aktivira
  • Velocity Switched - element aktivira se samo pri određenim velocity razinama

Layer i Split postavke

Layer Voice postavke

Kada kreirate Layer zvuk (dva zvuka zajedno):

  • Voice 1 / Voice 2 - odabir dva zvuka za kombinaciju
  • Volume - glasnoća svakog zvuka
  • Pan - pozicija svakog zvuka (stereo širina)
  • Octave - transpozicija svakog zvuka gore/dolje
  • Key Range - raspon tipkovnice gdje se svaki zvuk aktivira

Split Voice postavke

Za Split zvukove:

  • Split Point - točka podjele tipkovnice (npr. C4)
  • Lower Voice - zvuk za lijevi dio
  • Upper Voice - zvuk za desni dio
  • Split Point Voice - zvuk za samu točku podjele

Efekti u Voice Editoru

Dostupni efekti

Yamaha klavijature imaju brojne efekte:

  • Reverb - prostorni efekt (Room, Hall, Plate, itd.)
  • Chorus - dubina i širina
  • Delay - echo efekt
  • DSP Effects - dodatni efekti (distortion, wah, flange, itd.)

Send vs Insert efekti

Send efekti (Reverb, Chorus) - primjenjuju se paralelno, možete kontrolirati koliko se primjenjuje

Insert efekti (DSP) - primjenjuju se direktno na zvuk, potpuno ga mijenjaju

Praktični primjeri

Primjer 1: "Bright Piano"

Svjetliji, oštriji piano zvuk:

  1. Odaberite Grand Piano
  2. Voice Edit → Element 1
  3. Povećajte Filter Cutoff na 110-127
  4. Smanjite Resonance na 20-30
  5. Povećajte Attack malo (5-10) za mekši početak
  6. Spremite kao "Bright Piano"

Primjer 2: "String Pad s Vibrato"

Strings s LFO vibrato efektom:

  1. Odaberite Strings zvuk
  2. Voice Edit → Element
  3. Idite na LFO postavke
  4. Waveform: Sine
  5. Speed: 30-40 (umjereno vibrato)
  6. Modulation Target: Pitch
  7. Depth: 20-30
  8. Delay: 20 (vibrato počinje nakon malog odgode)
  9. Spremite

Primjer 3: "Fat Lead Synth"

Debeli, puni synth zvuk:

  1. Odaberite Lead Synth zvuk
  2. Voice Edit → Layer
  3. Koristite isti zvuk za Voice 1 i Voice 2
  4. Voice 1: Volume 100%, Pan L30
  5. Voice 2: Volume 100%, Pan R30, Detune +5
  6. Dodajte malo Chorus
  7. Spremite kao "Fat Lead"

Koristan savjeti

Snimite svoje postavke

Prije nego što počnete eksperimentirati, zapišite ili snimite originalne postavke. Tako možete uvijek vratiti se ako nešto pođe po zlu.

Testiranje u kontekstu

Uvijek testirajte svoj prilagođeni zvuk s stilom ili u kombinaciji s drugim zvukovima. Ono što zvuči dobro solo možda neće raditi u miks.

Koristite Reference zvukove

Uspoređujte svoje kreacije s profesionalnim zvukovima iz preseta. To će vam pomoći razumjeti kako različite postavke utječu na zvuk.

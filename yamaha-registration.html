Registration Memory

Registration Memory je jedna od najkorisnijih funkcija Yamaha klavijatura. Omogućava vam da spremite kompletan set postavki i instantno ih učitavate jednim pritiskom.

Što je Registration Memory?

Koncept

Registration Memory omogućava vam da spremite sve postavke vašeg instrumenta u jedan "snimak":

  • Zvukovi (Right 1, Right 2, Left)
  • Stilovi i njihove postavke
  • Glasnoće i balans
  • Efekti (Reverb, Chorus, DSP)
  • Multi-Pad postavke
  • Split i Layer postavke
  • I mnogo više

Zašto koristiti Registration?

Prednosti:

  • Brza promjena - prijelaz između različitih postavki jednim pritiskom
  • Organizacija - grupirajte postavke po pjesmama ili koncertima
  • Profesionalnost - pripremite sve prije nastupa
  • Backup - sigurnost vaših postavki

Kako funkcionira Registration?

Registration Bank

Svaka Registration memorija sastoji se od Bank-a:

  • Bank - skupina od 8 Registration slotova
  • Slot - pojedinačna Registration (obično 1-8)
  • Možete imati više Bank-ova (A, B, C, itd.)

Primjer: Bank A, Slot 1-8 = 8 različitih setova postavki

Tipke za Registration

Na većini Yamaha klavijatura imate:

  • REGISTRATION - pristup Registration memoriji
  • Memory Bank tipke (A, B, C, itd.) - odabir banka
  • 1-8 tipke - odabir slotova

Kreiranje Registration-a - korak po korak

1. Pripremite sve postavke

Prije spremanja, postavite sve kako želite:

  1. Odaberite Right Voice (1 i/ili 2)
  2. Odaberite Left Voice (ako koristite)
  3. Odaberite Style
  4. Podesite glasnoće za svaki element
  5. Podesite efekte
  6. Odaberite Multi-Pad (ako koristite)
  7. Podesite sve ostale željene postavke

2. Otvorite Registration

  1. Pritisnite REGISTRATION tipku
  2. Odaberite Bank (A, B, C, itd.)
  3. Odaberite prazan Slot (1-8)

3. Spremite Registration

Različiti modeli imaju različite metode:

  • Metoda 1: Pritisnite i držite Slot tipku (1-8) dok se registrira
  • Metoda 2: Pritisnite MENU → Save Registration
  • Metoda 3: Dodirnite "Save" na touchscreen-u

Dajte Registration-u opisno ime i potvrdite spremanje.

Koristan savjet - Imenovanje

Koristite opisna imena za vaše Registration-e. Na primjer:

  • "Ballad_Piano_Strings"
  • "Rock_Band_Style"
  • "Jazz_Trio_Setup"

To će vam pomoći brzo pronaći ono što tražite.

Korištenje Registration-a

Učitavanje Registration-a

  1. Pritisnite REGISTRATION tipku
  2. Odaberite Bank (A, B, C, itd.)
  3. Pritisnite Slot tipku (1-8) za željeni Registration
  4. Sve postavke će se automatski primijeniti!

Tijekom nastupa

Registration omogućava brze prijelaze:

  • Za svaku pjesmu možete imati vlastiti Registration
  • Prelazite između verzija iste pjesme (verse, chorus, solo)
  • Promijenite cijeli set postavki jednim pritiskom

Napredne Registration postavke

Što se sprema u Registration?

Ovisno o vašoj klavijaturi, Registration može spremiti:

  • ✓ Voice postavke (Right 1, Right 2, Left)
  • ✓ Style postavke i odabrani stil
  • ✓ Volume i Balance postavke
  • ✓ Effect postavke (Reverb, Chorus, DSP)
  • ✓ Split Point (ako je postavljen)
  • ✓ Layer postavke
  • ✓ Multi-Pad postavke
  • ✓ Tempo
  • ✓ Transpose postavke
  • ✓ Harmony/Echo postavke
  • ✓ I mnogo više...

Selektivno spremanje

Na nekim modelima možete odabrati što želite spremiti u Registration. To je korisno ako želite spremiti samo određene postavke.

Registration Link

Napredna funkcija koja omogućava:

  • Automatsko učitavanje određenih Registration-a s određenim stilovima
  • Učitavanje različitih Registration-a za različite sekcije pjesme

Organizacija Registration-a

Organizacija po koncertima

Kreirajte Bank za svaki koncert:

  • Bank A - Koncert 1 (Pjesma 1-8)
  • Bank B - Koncert 2 (Pjesma 1-8)
  • itd.

Organizacija po stilovima

Grupirajte po muzičkim stilovima:

  • Bank A - Ballad postavke
  • Bank B - Rock postavke
  • Bank C - Jazz postavke
  • itd.

Koristan savjet - Redoslijed

Organizirajte Registration-e u redoslijedu u kojem ćete ih koristiti tijekom nastupa. Tako možete jednostavno prelaziti sa Slot 1 → Slot 2 → Slot 3...

Spremanje i prijenos

Spremanje na USB

Vaše Registration-e možete spremiti na USB:

  1. Pritisnite MENU → File ili Utility
  2. Odaberite Save Registration
  3. Odaberite Registration Bank koji želite spremiti
  4. Odaberite USB kao destinaciju
  5. Potvrdite spremanje

Format: .rgt datoteke (Registration datoteke)

Učitavanje s USB-a

Možete učitati Registration-e s USB memorije:

  1. Povežite USB memoriju
  2. MENU → File → Load Registration
  3. Odaberite .rgt datoteku
  4. Odaberite destinaciju Bank
  5. Potvrdite učitavanje

Dijeljenje Registration-a

Možete dijeliti svoje Registration-e s drugim Yamaha korisnicima spremajući ih na USB i dijeleći .rgt datoteke.

Koristan savjeti

Backup vaših Registration-a

Uvijek napravite backup vaših Registration-a na USB. Ako se nešto dogodi s klavijaturom, nećete izgubiti sve svoje postavke.

Testiranje Registration-a

Prije nastupa, prođite kroz sve vaše Registration-e i provjerite da sve radi kako treba. Provjerite glasnoće, stilove i sve postavke.

Koristite Registration za učenje

Spremite različite postavke za različite situacije (solo sviranje, prateći, itd.) i koristite ih dok vježbate.

Kreiranje "Master" Registration-a

Kreirajte Registration s vašim favorit postavkama kao početnu točku za nove kreacije:

  • Vaši omiljeni zvukovi
  • Standardne postavke glasnoće
  • Vaši omiljeni efekti
  • Standardni stil

Koristite ga kao template za nove Registration-e.

Izrada Tone-ova na Roland klavijaturama

Roland koristi Tone umjesto "Voice" za zvukove. Tone-ovi na Roland klavijaturama koriste različite sintesne tehnologije (SuperNATURAL ili ZEN-Core) i imaju specifičan način kreiranja koji se razlikuje od Yamaha pristupa.

Što je Tone?

Definicija

Tone je pojedinačni zvuk instrumenta koji možete svirati. Svaki Tone može biti:

  • Elementarni - jednostavni zvuk jedne vrste instrumenta
  • Layer - kombinacija više Tone-ova koji se sviraju istovremeno
  • Split - različiti Tone-ovi za različite dijelove tipkovnice
  • Multi-sampled - koristi više audio uzoraka za različite note

Roland sintesne tehnologije

SuperNATURAL (BK-3, BK-5):

  • Koristi .SVN format
  • Realistični zvukovi instrumenata
  • Visoka kvaliteta zvuka

ZEN-Core (BK-7m, EA-7, noviji modeli):

  • Koristi .SVZ format
  • Više fleksibilnosti u kreiranju
  • Kompatibilnost s različitim Roland produktima
  • Napredne mogućnosti editiranja

Kako funkcionira Tone?

Osnovni principi

Roland Tone-ovi rade na principu:

  1. Oscillator (Osc) - generira osnovni zvuk (waveform ili sample)
  2. Filter - oblikuje karakter zvuka
  3. Amp Envelope - kontrolira glasnoću kroz vrijeme
  4. LFO - dodaje modulacije i pokretanje
  5. Efekti - reverb, chorus, delay, itd.

Struktura Tone-a

Tone može sastojati od:

  • Single Tone - jedan osnovni zvuk
  • Layer Tone - dva ili više Tone-ova istovremeno
  • Split Tone - različiti Tone-ovi za različite zone tipkovnice
  • Parts - Tone može imati više Parts (ZEN-Core)

Tipovi Tone-ova

1. Single Tone (Pojedinačni Tone)

Osnovni tip Tone-a - jedan instrument u cijelom rasponu tipkovnice.

Primjer: Grand Piano, Strings, Trumpet

2. Layer Tone (Slojevi)

Dva ili više Tone-ova sviraju istovremeno:

  • Dual Layer - dva Tone-a zajedno (npr. Piano + Strings)
  • Triple Layer - tri Tone-a
  • Quad Layer - četiri Tone-a (na modelima koji podržavaju)

Kako kreirati: Tone Edit → Layer → odaberite Tone-ove i podešavanja

3. Split Tone (Podijeljeni Tone)

Različiti Tone-ovi za različite dijelove tipkovnice:

  • Split Point - točka gdje se Tone mijenja
  • Lower - lijevi dio (obično bas)
  • Upper - desni dio (obično melodija)

Primjer: Bas lijevo, Piano desno

4. Multi-sampled Tone

Koristi više audio uzoraka za različite note:

  • Različiti sample-i za različite oktave
  • Prirodniji zvuk kroz cijeli raspon
  • Više memorije, ali bolja kvaliteta

Kreiranje vlastitog Tone-a

Metoda 1: Modifikacija postojećeg Tone-a

Najlakši način za početak:

  1. Odaberite Tone koji vam je najbliži željenom
  2. Pritisnite "MENU" → "Tone Edit" ili "TONE EDIT"
  3. Izmijenite parametre:
    • Filter (cutoff, resonance)
    • Envelope (attack, decay, sustain, release)
    • LFO (brzina, dubina, waveform)
    • Efekte (reverb, chorus, delay)
    • Glasnoću i pan
  4. Spremite kao novi User Tone

Metoda 2: Kreiranje Layer Tone-a

Kombinacija više Tone-ova:

  1. Tone Edit → Layer
  2. Odaberite prvi Tone
  3. Odaberite drugi Tone
  4. Podesite glasnoću svakog Tone-a
  5. Podesite panoramu (pan) za stereo efekt
  6. Podesite oktaver ako je potrebno
  7. Dodajte efekte
  8. Spremite

Savjet: Kombinirajte komplementarne Tone-ove. Na primjer, Piano (bogati u srednjim frekvencijama) + Strings (bogati u visokim) = puni, ljepši zvuk.

Metoda 3: Split Tone kreiranje

Različiti Tone-ovi za različite zone:

  1. Tone Edit → Split
  2. Postavite Split Point (točka podjele)
  3. Odaberite Lower Tone (lijevo)
  4. Odaberite Upper Tone (desno)
  5. Podesite glasnoću za svaki dio
  6. Podesite oktaver ako je potrebno
  7. Dodajte efekte
  8. Spremite

Metoda 4: Kreiranje iz sample-a

Roland klavijature omogućavaju kreiranje Tone-ova iz WAV sample-a:

  1. Pripremite WAV sample-e
  2. Učitajte ih u klavijaturu
  3. Tone Edit → Sample
  4. Dodijelite sample-e različitim notama
  5. Podesite parametre (Filter, EG, itd.)
  6. Za loop sample-e, uključite Loop
  7. Spremite kao novi Tone

ZEN-Core specifičnosti

ZEN-Core Parts

ZEN-Core Tone-ovi (noviji modeli) mogu imati više Parts:

  • Part - pojedinačni zvuk u Tone-u
  • Tone može imati više Parts - što više Parts, kompleksniji Tone
  • Svaki Part ima vlastiti Oscillator, Filter, EG, LFO
  • Part Mix - kontrola kako se Parts miješaju

ZEN-Core oscilatori

ZEN-Core Tone-ovi mogu koristiti:

  • PCM Wave - uzorkovani zvukovi
  • Virtual Analog - sintetizirani zvukovi
  • Wavetable - wavetable sintesu (neki modeli)

Praktični primjeri

Primjer 1: "Warm Piano"

Kreiranje toplijeg, mekšeg piano zvuka:

  1. Odaberite "Grand Piano" Tone
  2. Tone Edit → Common ili Filter
  3. Smanjite Filter Cutoff (tamniji zvuk)
  4. Povećajte Attack malo (meksi početak)
  5. Povećajte Release za duži tail
  6. Dodajte malo Reverb
  7. Spremite kao "Warm Piano"

Primjer 2: "Pad + Lead" Layer

Kombinacija pozadinskog i vodećeg zvuka:

  1. Tone Edit → Layer
  2. Odaberite "Strings" kao prvi Tone (glasnoća 70%)
  3. Odaberite "Lead Synth" kao drugi Tone (glasnoća 100%)
  4. Podesite pan: Strings lijevo (-30), Lead centar
  5. Strings: dodajte Chorus
  6. Lead: dodajte malo Delay
  7. Spremite

Primjer 3: "Bass + Piano" Split

Bas lijevo, Piano desno:

  1. Tone Edit → Split
  2. Postavite Split Point na C4 (sredina)
  3. Lower Tone: "Electric Bass"
  4. Upper Tone: "Grand Piano"
  5. Podesite glasnoće (Bas možda malo jači)
  6. Spremite

Spremanje i organizacija

User Tone memorija

Vaši prilagođeni Tone-ovi spremaju se u:

  • User Tone Bank - interno spremište klavijature
  • USB - za backup i prijenos

Format: .SVZ (ZEN-Core) ili .SVN (SuperNATURAL) datoteke

Organizacija

Organizirajte svoje Tone-ove po kategorijama i dodajte opisna imena. Na primjer:

  • "Piano_Warm_Custom"
  • "Layer_Strings_Pad"
  • "Split_Bass_Piano"

Backup vaših Tone-ova

Uvijek napravite backup vaših prilagođenih Tone-ova na USB memoriju. Ako se nešto dogodi s klavijaturom ili resetirate postavke, nećete izgubiti svoj rad.

Eksperimentiranje

Ne bojte se eksperimentirati! Najbolji Tone-ovi često nastaju slučajno. Podesite parametre i slušajte što se događa.

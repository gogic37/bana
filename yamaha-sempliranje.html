Sempliranje na Yamaha klavijaturama

Sempliranje (sampling) omogućava vam da snimite zvukove i koristite ih kao dijelove vaših zvukova, stilova ili Multi-Pad-ova. Ovo vam daje beskonačne mogućnosti za kreiranje jedinstvenih zvukova.

Što je sempliranje?

Definicija

Sample je digitalni zapis zvuka. Na Yamaha klavijaturama možete:

  • Snimiti vlastite zvukove (mikrofon, linijski ulaz)
  • Učitati audio datoteke
  • Koristiti sample-e u Voice-ovima
  • Koristiti sample-e u Style-ovima
  • Koristiti sample-e kao Multi-Pad-ove

Zašto semplirati?

  • Jedinstveni zvukovi - dodajte vlastite zvukove koje ne možete pronaći u preset-u
  • Personalizacija - koristite svoje glasove, zvukove instrumenata, itd.
  • Kreativnost - kombinujte različite zvukove u nove kreacije
  • Profesionalnost - dodajte profesionalne sample-e u svoje kreacije

Kako funkcionira sempliranje?

Osnovni princip

Yamaha klavijature snimaju audio signal i pretvaraju ga u digitalni format:

  1. Snimanje - audio signal se snima (mikrofon, linijski ulaz, audio datoteka)
  2. Obrada - možete obrezati, normalizirati, dodati efekte
  3. Spremanje - sample se sprema u memoriju klavijature
  4. Korištenje - koristi se u Voice-u, Style-u ili Multi-Pad-u

Tipovi sample-a

  • Mono Sample - jednostavni zvuk (npr. udarac bubnja)
  • Stereo Sample - stereo zvuk s lijevim i desnim kanalom
  • Loop Sample - sample koji se ponavlja (koristan za padove, strings)
  • One-shot Sample - svira jednom i prestaje

Snimanje sample-a

Pristup Sample Editoru

  1. Pritisnite FUNCTION → Sample ili Audio Recorder
  2. Odaberite Record New Sample
  3. Prije nego što počnete, provjerite postavke:

1. Izvor zvuka

Odaberite izvor zvuka koji želite snimiti:

  • Mikrofon - povezani mikrofon
  • Audio In - linijski ulaz (audio interface, drugi instrument)
  • USB Audio - audio s USB memorije
  • Internal - zvuk same klavijature

2. Postavke snimanja

Podesite parametre prije snimanja:

  • Sample Rate - kvaliteta snimke (44.1kHz, 48kHz)
    • Viša = bolja kvaliteta, ali više memorije
    • 44.1kHz je dovoljno za većinu primjena
  • Bit Depth - dubina bita (16-bit, 24-bit)
    • 24-bit = bolja kvaliteta, ali više memorije
  • Format - Mono ili Stereo
    • Mono = manje memorije, dovoljno za jednostavne zvukove
    • Stereo = više memorije, ali potpuniji zvuk
  • Input Level - glasnoća ulaznog signala
    • Postavite tako da signal ne ide u crveno (clipping)
    • Zelena/zuta zona = dobro

Važno - Input Level

Pazite na input level! Previše glasan signal će stvoriti distorziju (clipping). Premalo će rezultirati slabo kvalitetnim snimkom s puno šuma. Podesite tako da je signal u optimalnoj zoni.

3. Snimanje

  1. Pritisnite Record da počnete snimanje
  2. Reproducirajte zvuk koji želite snimiti
  3. Pratite trajanje i memoriju
  4. Pritisnite Stop kada završite

Napomena: Provjerite koliko memorije imate dostupno. Sample-i mogu zauzeti puno prostora!

4. Obrada sample-a

Nakon snimanja, možete obraditi sample:

  • Trim/Trim Start/End - uklonite tišinu na početku/kraju
  • Normalize - automatsko povećanje glasnoće do maksimuma bez distorzije
  • Fade In/Out - glatko povećanje/smanjenje glasnoće
  • Reverse - okrenite sample unatrag
  • Time Stretch - promijenite duljinu bez promjene visine
  • Pitch Shift - promijenite visinu bez promjene duljine

Kreiranje loop sample-a

Što je loop?

Loop je sample koji se ponavlja bez prekida. Koristan za:

  • Pad zvukove (strings, synth pads)
  • Background zvukove
  • Sustavne elemente u stilovima

Kako kreirati dobar loop

  1. Snimite sample koji ima konstantnu karakteristiku
  2. Pronađite točku gdje se sample može glatko ponavljati
  3. Postavite Loop Start i Loop End točke
  4. Testirajte loop - trebao bi se glatko ponavljati bez "click" zvuka
  5. Podesite Crossfade ako je dostupno za glatki prijelaz

Savjet za dobar loop

Za najbolje rezultate, snimite sample koji već ima prirodnu ponavljajuću strukturu. Izbjegavajte sample-e s naglim promjenama jer će stvoriti "click" na spoju.

Korištenje sample-a u Voice-ovima

Kreiranje User Voice s Sample-om

  1. Snimite ili učitajte sample
  2. Voice Edit → Element
  3. Odaberite Waveform ili Sample opciju
  4. Odaberite svoj sample
  5. Podesite parametre (Filter, EG, itd.)
  6. Za loop sample-e, uključite Loop
  7. Podesite Note Range ako je potrebno
  8. Spremite kao novi User Voice

Multi-sample Voice

Za prirodniji zvuk, možete koristiti više sample-a za različite note:

  • Različiti sample-i za različite oktave
  • Velocity switched sample-i (različiti za lagane/jake pritiske)
  • Round robin sample-i (različiti za svaki pritisak)

Korištenje sample-a u Style-ovima

Dodavanje sample-a u Style

Sample-i se mogu koristiti u Style-ovima kao:

  • Style Parts - dodajte sample kao novi Style Part
  • One-shot - kratki zvukovi (udarci, efekti)
  • Loop - pozadinski padovi ili atmosfera

Postupak:

  1. Style Edit → Style Part
  2. Odaberite prazan Part
  3. Odaberite svoj sample
  4. Podesite note i timing
  5. Spremite stil

Multi-Pad s Sample-ima

Kreiranje Multi-Pad s Sample-ima

Multi-Pad-ovi su kratki audio uzorci koji se aktiviraju tipkom:

  1. Snimite ili učitajte sample-e
  2. Multi-Pad → Edit
  3. Odaberite Pad slot (obično 4-16 padova)
  4. Dodijelite sample svakom Pad-u
  5. Podesite glasnoću i efekte
  6. Spremite Multi-Pad set

Učitavanje audio datoteka

Podržani formati

Većina Yamaha klavijatura podržava:

  • WAV - najbolja kvaliteta, bez kompresije
  • AIFF - Mac format, visoka kvaliteta
  • MP3 - komprimirani format (neki modeli)

Učitavanje s USB-a

  1. Kopirajte audio datoteku na USB memoriju
  2. Umetnite USB u klavijaturu
  3. Sample Editor → Load
  4. Odaberite datoteku s USB-a
  5. Datoteka će se učitati kao sample
  6. Možete ga obrađivati kao i snimljeni sample

Upravljanje memorijom

Memorija

Sample-i mogu zauzeti puno memorije! Pazite na:

  • Duljinu sample-a (kraći = manje memorije)
  • Sample rate (niži = manje memorije)
  • Format (Mono = manje memorije od Stereo)

Primjer: 44.1kHz, 16-bit, Mono, 10 sekundi = ~880 KB

Optimizacija memorije

  • Koristite Mono umjesto Stereo kada je moguće
  • Smanjite Sample Rate ako kvaliteta dozvoljava (32kHz umjesto 44.1kHz)
  • Obrezite nepotrebne dijelove sample-a
  • Brišite neiskorištene sample-e

Koristan savjeti

Kvalitetna snimka

Za najbolju kvalitetu:

  • Snimajte u tihoj prostoriji (manje pozadinske buke)
  • Koristite kvalitetan mikrofon ili linijski ulaz
  • Podesite input level da ne ide u crveno
  • Normalizujte sample nakon snimanja

Backup vaših sample-a

Vaše sample-e možete spremiti na USB. Organizirajte ih u mape i napravite backup. Tako možete koristiti iste sample-e na različitim klavijaturama ili nakon resetiranja.

Eksperimentirajte

Ne bojte se eksperimentirati s različitim sample-ima. Kombinirajte neočekivane zvukove - često ćete dobiti najzanimljivije rezultate!

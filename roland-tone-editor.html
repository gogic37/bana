Tone Editor - Napredno uređivanje Tone-ova

Tone Editor je moćan alat za napredno kreiranje i prilagođavanje Tone-ova na Roland klavijaturama. Omogućava vam detaljnu kontrolu nad svakim aspektom zvuka.

Pristup Tone Editoru

Kako otvoriti Tone Editor

  1. Odaberite Tone koji želite urediti (ili kopirati)
  2. Pritisnite MENU → Tone Edit ili TONE EDIT
  3. Alternativno: Pritisnite EDIT ako je dostupan na vašoj klavijaturi

Napomena: Neki preset Tone-ovi ne mogu se direktno mijenjati - morate ih prvo kopirati u User zone.

Struktura Tone Editora

Osnovni ekrani

Tone Editor obično ima nekoliko kategorija/ekrana:

  • Common - zajednički parametri za cijeli Tone
  • Oscillator - osnovni zvuk (waveform, sample)
  • Filter - oblikovanje zvuka
  • Amp/Envelope - kontrola glasnoće
  • LFO - modulacije
  • Effect - detaljne postavke efekata
  • Layer - postavke za Layer tipove
  • Split - postavke za Split tipove

Common parametri

Volume (Glasnoća)

Ukupna glasnoća Tone-a (0-127).

Savjet: Postavite na 100-120 za optimalnu glasnoću bez distorzije.

Pan (Panorama)

Pozicija Tone-a u stereo prostoru:

  • L64 - krajnje lijevo
  • C - centar
  • R63 - krajnje desno

Reverb Send

Koliko reverb efekta primjenjuje se na Tone (0-127).

Chorus Send

Koliko chorus efekta primjenjuje se (0-127).

Oscillator parametri

Waveform/Wave

Osnovni zvuk Oscillator-a može biti:

  • PCM Wave - uzorkovani zvukovi (realni instrumenti)
  • Virtual Analog - sintetizirani zvukovi
  • Wavetable - wavetable sintesu (ZEN-Core, neki modeli)

Napomena: Na nekim modelima možete mijenjati waveform, na drugima ne (ovo ovisi o Tone-u).

Tuning

Fino podešavanje visine tona:

  • Coarse Tune - grubo podešavanje (u semitonima)
  • Fine Tune - fino podešavanje (u centima)

Filter parametri

Filter Type

Tip filtera:

  • Low Pass - propušta niske frekvencije (najčešći)
  • High Pass - propušta visoke frekvencije
  • Band Pass - propušta srednje frekvencije
  • Band Reject - blokira srednje frekvencije

Cutoff Frequency

Točka odsijecanja filtera (0-127):

  • Niska vrijednost = tamniji, mutniji zvuk
  • Visoka vrijednost = svjetliji, oštriji zvuk

Resonance

Naglasak na cutoff frekvenciji (0-127):

  • Viša vrijednost = zvižduk na cutoff točki
  • Koristi se za efekte poput wah-wah

Envelope Generator (EG)

Amp Envelope

Kontrolira kako se glasnoća mijenja kroz vrijeme:

  • Attack Time (0-127) - koliko brzo zvuk dosegne punu glasnoću
    • 0 = trenutno (udarac)
    • 127 = vrlo sporo
  • Decay Time (0-127) - koliko brzo padne nakon attack-a
  • Sustain Level (0-127) - razina dok držite tipku
  • Release Time (0-127) - koliko brzo se zvuk gasi nakon puštanja

Filter Envelope

Kontrolira kako se filter mijenja kroz vrijeme:

  • Možete kontrolirati cutoff frequency s envelope-om
  • Kreira dinamične filter efekte
  • Attack, Decay, Sustain, Release parametri

ADSR primjeri

Piano: Kratak Attack (10-20), srednji Decay, visok Sustain, srednji Release

Strings: Srednji Attack (30-50), kratak Decay, visok Sustain, dug Release (80-100)

Percussion: Trenutni Attack (0-5), kratak Decay, nizak Sustain, kratak Release

LFO (Low Frequency Oscillator)

LFO parametri

Dodaje oscilacije i pokretanje zvuku:

  • Waveform - oblik oscilacije (sine, triangle, square, sawtooth, random)
  • Speed/Rate - brzina oscilacije
  • Delay - odgoda prije početka oscilacije
  • Fade In - postupno povećanje oscilacije
  • Depth - dubina modulacije

LFO Modulation Target

Što LFO modulira:

  • Filter Cutoff - wah-wah efekt
  • Pitch - vibrato
  • Amplitude - tremolo
  • Pan - auto-pan efekt

ZEN-Core specifičnosti

Parts u ZEN-Core Tone-u

ZEN-Core Tone-ovi mogu imati više Parts:

  • Svaki Part ima vlastiti Oscillator, Filter, EG, LFO
  • Parts se miješaju zajedno
  • Možete kontrolirati Volume i Pan svakog Part-a
  • Parts mogu imati različite Key Range

Part Mix

Kontrola miješanja Parts:

  • Volume - glasnoća svakog Part-a
  • Pan - pozicija svakog Part-a
  • Send Level - koliko efekta prima svaki Part

Praktični primjeri

Primjer 1: "Bright Piano"

Svjetliji, oštriji piano zvuk:

  1. Odaberite Grand Piano Tone
  2. Tone Edit → Filter
  3. Povećajte Filter Cutoff na 110-127
  4. Smanjite Resonance na 20-30
  5. Povećajte Attack malo (5-10)
  6. Spremite kao "Bright Piano"

Primjer 2: "String Pad s Vibrato"

Strings s LFO vibrato efektom:

  1. Odaberite Strings Tone
  2. Tone Edit → LFO
  3. Waveform: Sine
  4. Speed: 30-40
  5. Modulation Target: Pitch
  6. Depth: 20-30
  7. Delay: 20
  8. Spremite

Koristan savjet

Snimite svoje postavke prije nego što počnete eksperimentirati. Tako možete uvijek vratiti se na original ako nešto pođe po zlu.

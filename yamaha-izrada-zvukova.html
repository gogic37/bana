Izrada zvukova na Yamaha klavijaturama

Yamaha zvukovi (Voices) su osnova vašeg sviranja. Razumijevanje kako funkcioniraju i kako ih kreirati omogućit će vam da postignete jedinstvene zvučne rezultate.

Što je zvuk (Voice)?

Definicija

Voice je pojedinačni zvuk instrumenta koji možete svirati. Svaki zvuk može biti:

  • Elementarni - jednostavni zvuk jedne vrste instrumenta
  • Layer - kombinacija više zvukova koji se sviraju istovremeno
  • Split - različiti zvukovi za različite dijelove tipkovnice
  • Dual - dva zvuka u istoj oktavi (dupli zvuk)

Kako funkcionira zvuk?

Osnovni principi

Yamaha zvukovi rade na principu:

  1. Waveform (valni oblik) - osnovni zvuk uzorkovan ili sintetiziran
  2. Filter - oblikuje karakter zvuka
  3. Amplifier (Envelope) - kontrolira glasnoću kroz vrijeme
  4. Efekti - reverb, chorus, delay, itd.

Elementi zvuka

Svaki zvuk sastoji se od elemenata (Elements):

  • Element - pojedinačni zvuk koji čini Voice
  • Voice može imati 1-4 elementa - što više elemenata, kompleksniji zvuk
  • Svaki element ima vlastiti waveform, filter, envelope i efekte

Tipovi zvukova

1. Normal Voice (Standardni zvuk)

Osnovni tip zvuka - jedan instrument u cijelom rasponu tipkovnice.

Primjer: Grand Piano, Strings, Trumpet

2. Layer Voice (Slojevi)

Dva ili više zvukova sviraju istovremeno:

  • Dual - dva zvuka zajedno (npr. Piano + Strings)
  • Triple - tri zvuka
  • Quad - četiri zvuka

Kako kreirati: Voice Edit → Layer → odaberite zvukove i podešavanja

3. Split Voice (Podijeljeni zvuk)

Različiti zvukovi za različite dijelove tipkovnice:

  • Split Point - točka gdje se zvuk mijenja
  • Lower - lijevi dio (obično bas)
  • Upper - desni dio (obično melodija)

Primjer: Bas lijevo, Piano desno

4. Left Voice (Lijevi zvuk)

Zvuk koji se svira lijevom rukom, neovisno o stilu:

  • Koristi se za bas linije ili akorde
  • Radi nezavisno od Right Voice-a

Kreiranje vlastitog zvuka

Metoda 1: Modifikacija postojećeg zvuka

Najlakši način za početak:

  1. Odaberite zvuk koji vam je najbliži željenom
  2. Pritisnite "FUNCTION" → "Voice Edit"
  3. Izmijenite parametre:
    • Filter (cutoff, resonance)
    • Envelope (attack, decay, sustain, release)
    • Efekte (reverb, chorus)
    • Glasnoću svakog elementa
  4. Spremite kao novi User Voice

Metoda 2: Kreiranje Layer zvuka

Kombinacija više zvukova:

  1. Voice Edit → Layer
  2. Odaberite prvi zvuk
  3. Odaberite drugi zvuk
  4. Podesite glasnoću svakog zvuka
  5. Podesite panoramu (pan) za stereo efekt
  6. Dodajte efekte
  7. Spremite

Savjet: Kombinirajte komplementarne zvukove. Na primjer, Piano (bogati u srednjim frekvencijama) + Strings (bogati u visokim) = puni, ljepši zvuk.

Metoda 3: Split Voice kreiranje

Različiti zvukovi za različite zone:

  1. Voice Edit → Split
  2. Postavite Split Point (točka podjele)
  3. Odaberite Lower Voice (lijevo)
  4. Odaberite Upper Voice (desno)
  5. Podesite glasnoću za svaki dio
  6. Dodajte efekte
  7. Spremite

Voice Editor - detaljno

Common parametri (Zajednički)

Parametri koji utječu na cijeli zvuk:

Parametar Opis
Volume Ukupna glasnoća zvuka
Pan Pozicija u stereo prostoru (lijevo/desno)
Reverb Send Koliko reverb efekta primjenjuje se
Chorus Send Koliko chorus efekta primjenjuje se

Element parametri

Svaki element zvuka ima:

  • Waveform - osnovni zvuk (može se mijenjati ako je moguće)
  • Filter Cutoff - odsijecanje frekvencija (svjetlije/tamnije)
  • Filter Resonance - naglasak na cutoff frekvenciji
  • EG (Envelope Generator):
    • Attack - koliko brzo zvuk dosegne punu glasnoću
    • Decay - koliko brzo padne nakon attack-a
    • Sustain - razina dok držite tipku
    • Release - koliko brzo se zvuk gasi nakon puštanja tipke
  • Volume - glasnoća elementa
  • Detune - fino podešavanje visine tona

Praktični primjeri

Primjer 1: "Warm Piano"

Kreiranje toplijeg, mekšeg piano zvuka:

  1. Odaberite "Grand Piano"
  2. Voice Edit → Element
  3. Smanjite Filter Cutoff (tamniji zvuk)
  4. Povećajte Attack malo (meksi početak)
  5. Dodajte malo Reverb
  6. Spremite kao "Warm Piano"

Primjer 2: "Pad + Lead" Layer

Kombinacija pozadinskog i vodećeg zvuka:

  1. Voice Edit → Layer
  2. Odaberite "Strings" kao prvi zvuk (glaznoća 70%)
  3. Odaberite "Lead Synth" kao drugi zvuk (glasnoća 100%)
  4. Podesite pan: Strings lijevo (-30), Lead centar
  5. Dodajte Chorus na Strings
  6. Spremite

Primjer 3: "Bass + Piano" Split

Bas lijevo, Piano desno:

  1. Voice Edit → Split
  2. Postavite Split Point na C4 (sredina)
  3. Lower Voice: "Electric Bass"
  4. Upper Voice: "Grand Piano"
  5. Podesite glasnoće
  6. Spremite

Napredne tehnike

Velocity Switched Elements

Različiti elementi aktiviraju se ovisno o tome koliko snažno pritisnete tipku:

  • Lagani pritisak → meki zvuk
  • Snažan pritisak → agresivniji zvuk

Omogućava vam prirodniji, dinamičniji zvuk.

Oscillator Sync

Napredna tehnika za sintetizirane zvukove - sinkronizacija više oscilatora za jedinstvene zvukove.

Filter Modulation

Modulacija filtera pomoću LFO (Low Frequency Oscillator) ili Envelopea stvara dinamične, živahne zvukove.

Koristan savjet - Experimentation

Ne bojte se eksperimentirati! Najbolji zvukovi često nastaju slučajno. Podesite parametre i slušajte što se događa. Ako nešto ne zvuči dobro, jednostavno vratite se na prethodne postavke.

Spremanje i organizacija

User Voice memorija

Vaši prilagođeni zvukovi spremaju se u:

  • User Voice Bank - interno spremište klavijature
  • USB - za backup i prijenos

Format: .vce ili .uvn datoteke (ovisno o modelu)

Organizacija

Organizirajte svoje zvukove po kategorijama i dodajte opisna imena. Na primjer:

  • "Piano_Warm_Custom"
  • "Layer_Strings_Pad"
  • "Split_Bass_Piano"

Backup vaših zvukova

Uvijek napravite backup vaših prilagođenih zvukova na USB memoriju. Ako se nešto dogodi s klavijaturom ili resetirate postavke, nećete izgubiti svoj rad.

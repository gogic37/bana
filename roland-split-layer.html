Split i Layer na Roland klavijaturama

Roland klavijature omogućavaju kombinaciju više Tone-ova kroz Split i Layer funkcije. Ovo vam daje ogromne mogućnosti za kreativno sviranje.

Split (Podijeljena tipkovnica)

Što je Split?

Split omogućava da različiti Tone-ovi sviraju na različitim dijelovima tipkovnice:

  • Split Point - točka gdje se Tone mijenja
  • Lower - lijevi dio tipkovnice (obično bas)
  • Upper - desni dio tipkovnice (obično melodija)

Primjer: Bas lijevo (Lower), Piano desno (Upper)

Kako postaviti Split

  1. Pritisnite "MENU" → "Split/Layer" ili "SPLIT"
  2. Odaberite "Split" mod
  3. Postavite Split Point (obično C4 ili drugi ton)
  4. Odaberite Lower Tone (lijevi dio)
  5. Odaberite Upper Tone (desni dio)
  6. Podesite glasnoću za svaki dio
  7. Spremite ako je potrebno

Split Point postavke

Split Point određuje gdje se tipkovnica dijeli:

  • Možete odabrati bilo koju notu kao Split Point
  • Lower dio uključuje Split Point, Upper počinje nakon njega
  • Split Point se može mijenjati u realnom vremenu

Savjet za Split Point

C4 (srednji C) je najčešći izbor jer je prirodna sredina tipkovnice. Međutim, možete koristiti bilo koju notu ovisno o vašim potrebama.

Layer (Slojevi)

Što je Layer?

Layer omogućava da dva ili više Tone-ova sviraju istovremeno na istom dijelu tipkovnice:

  • Dual Layer - dva Tone-a zajedno
  • Triple Layer - tri Tone-a (na modelima koji podržavaju)
  • Quad Layer - četiri Tone-a (na modelima koji podržavaju)

Primjer: Piano + Strings = puniji, bogatiji zvuk

Kako postaviti Layer

  1. Pritisnite "MENU" → "Split/Layer" ili "LAYER"
  2. Odaberite "Layer" mod
  3. Odaberite prvi Tone
  4. Odaberite drugi Tone
  5. Podesite glasnoću svakog Tone-a
  6. Podesite pan (panoramu) ako želite stereo efekt
  7. Podesite oktaver ako je potrebno
  8. Spremite ako je potrebno

Layer glasnoće

Balansirajte glasnoću između Layer Tone-ova:

  • Možete podesiti glasnoću svakog Tone-a zasebno
  • Obično jedan Tone je glavni, drugi dodaje boju
  • Primjer: Piano 100%, Strings 50%

Kombinacija Split i Layer

Split + Layer kombinacija

Roland klavijature omogućavaju kombinaciju obje funkcije:

  • Lower Split + Layer - Layer Tone-ovi u Lower dijelu
  • Upper Split + Layer - Layer Tone-ovi u Upper dijelu
  • Obje zone - Layer u oba dijela tipkovnice

Primjer: Kompleksna postavka

Možete kreirati vrlo kompleksne postavke:

  • Lower: Bas (jedan Tone)
  • Upper: Piano + Strings (Layer od 2 Tone-a)
  • Total: 3 Tone-a na cijeloj tipkovnici

Key Range (Raspon tipkovnice)

Key Range kontrola

Možete kontrolirati raspon tipkovnice gdje se svaki Tone aktivira:

  • Low Key - najniža nota gdje se Tone aktivira
  • High Key - najviša nota gdje se Tone aktivira
  • Korisno za napredne kombinacije

Praktični primjeri

Primjer 1: Klasični Split

Bas lijevo, Piano desno:

  1. Split Point: C4
  2. Lower: Electric Bass (glasnoća 90%)
  3. Upper: Grand Piano (glasnoća 100%)
  4. Perfektno za solo sviranje s bas linijom

Primjer 2: Layer za bogatiji zvuk

Piano + Strings:

  1. Layer mod
  2. Tone 1: Grand Piano (glasnoća 80%)
  3. Tone 2: Strings (glasnoća 60%)
  4. Pan: Piano centar, Strings malo lijevo (-20)
  5. Rezultat: Puni, bogati zvuk

Primjer 3: Kompleksna kombinacija

Split s Layer u Upper zoni:

  1. Split Point: C4
  2. Lower: Electric Bass
  3. Upper: Layer (Piano + Strings)
  4. Total: 3 Tone-a aktivna

Koristan savjet

Eksperimentirajte s različitim kombinacijama. Često najbolji zvukovi dolaze iz neočekivanih kombinacija Tone-ova!

Live Set s Split/Layer

Spremanje u Live Set

Vaše Split/Layer postavke možete spremiti u Live Set:

  1. Postavite Split ili Layer kako želite
  2. Pritisnite "LIVE SET"
  3. Odaberite slot
  4. Pritisnite "WRITE"
  5. Sve postavke će biti spremljene

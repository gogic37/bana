Sempliranje na Roland klavijaturama

Sempliranje (sampling) omogućava vam da snimite zvukove i koristite ih u vašim Tone-ovima ili Rhythm-ima. Roland klavijature imaju napredne mogućnosti sempliranja, posebno za kreiranje multi-sampled Tone-ova.

Što je sempliranje?

Definicija

Sample je digitalni zapis zvuka. Na Roland klavijaturama možete:

  • Snimiti vlastite zvukove (mikrofon, linijski ulaz)
  • Učitati WAV datoteke
  • Koristiti sample-e u Tone-ovima
  • Koristiti sample-e u Rhythm-ima
  • Kreirati multi-sampled Tone-ove

Zašto semplirati?

  • Jedinstveni zvukovi - dodajte vlastite zvukove
  • Multi-sampling - prirodniji zvuk kroz cijeli raspon
  • Personalizacija - koristite svoje glasove, zvukove instrumenata
  • Kreativnost - kombinujte različite zvukove

Roland pristup sempliranju

Multi-sampling

Roland posebno dobro podržava multi-sampling:

  • Različiti sample-i za različite note
  • Priojeđniji zvuk kroz cijeli raspon tipkovnice
  • Možete mapirati sample-e na određene note
  • Roland automatski interpolira između sample-a

Snimanje sample-a

Pristup Sample Editoru

  1. Pritisnite "MENU" → "Sample" ili "SAMPLE EDITOR"
  2. Odaberite "Record New Sample"
  3. Prije nego što počnete, provjerite postavke

1. Izvor zvuka

Odaberite izvor zvuka koji želite snimiti:

  • Mikrofon - povezani mikrofon
  • Audio In - linijski ulaz
  • USB Audio - audio s USB memorije
  • Internal - zvuk same klavijature

2. Postavke snimanja

Podesite parametre:

  • Sample Rate - 44.1kHz ili 48kHz (viša = bolja kvaliteta)
  • Bit Depth - 16-bit ili 24-bit
  • Format - Mono ili Stereo
  • Input Level - glasnoća ulaznog signala

Važno: Pazite na input level! Previše glasan signal će stvoriti distorziju.

3. Snimanje

  1. Pritisnite "Record" da počnete snimanje
  2. Reproducirajte zvuk koji želite snimiti
  3. Pratite trajanje i memoriju
  4. Pritisnite "Stop" kada završite

4. Obrada sample-a

Nakon snimanja, možete obraditi sample:

  • Trim - uklonite tišinu na početku/kraju
  • Normalize - automatsko povećanje glasnoće
  • Fade In/Out - glatko povećanje/smanjenje
  • Reverse - okrenite sample unatrag
  • Time Stretch - promijenite duljinu
  • Pitch Shift - promijenite visinu

Kreiranje multi-sampled Tone-a

Multi-sampling proces

Za prirodniji zvuk, snimite više sample-a za različite note:

  1. Snimite sample za različite note (npr. svaka 4. nota)
  2. Učitajte sve sample-e u Sample Editor
  3. Tone Edit → Sample Mapping
  4. Mapirajte svaki sample na odgovarajuće note
  5. Podesite Key Range za svaki sample
  6. Roland će automatski interpolirati između sample-a

Savjet za multi-sampling

Snimite sample-e za svaku 3-4 notu za prirodniji zvuk. Za još bolju kvalitetu, snimite svaku notu, ali to zahtijeva puno memorije.

Učitavanje WAV datoteka

Podržani formati

Roland klavijature podržavaju:

  • WAV - najbolja kvaliteta, bez kompresije
  • AIFF - Mac format, visoka kvaliteta

Preporučene postavke: 44.1kHz, 16-bit ili 24-bit

Učitavanje s USB-a

  1. Kopirajte WAV datoteku na USB memoriju
  2. Umetnite USB u klavijaturu
  3. Sample Editor → Load
  4. Odaberite datoteku s USB-a
  5. Datoteka će se učitati kao sample

Korištenje sample-a u Tone-ovima

Kreiranje Tone-a s Sample-om

  1. Snimite ili učitajte sample
  2. Tone Edit → Oscillator
  3. Odaberite "Sample" kao Oscillator source
  4. Odaberite svoj sample
  5. Podesite parametre (Filter, EG, LFO, itd.)
  6. Za loop sample-e, uključite Loop i postavite Loop Points
  7. Spremite kao novi User Tone

Loop sample-i

Za sample-e koji se ponavljaju (pad-ovi, strings):

  • Postavite Loop Start i Loop End točke
  • Testirajte loop - trebao bi se glatko ponavljati
  • Podesite Crossfade ako je dostupno
  • Uključite Loop u Tone Editoru

Upravljanje memorijom

Memorija

Sample-i mogu zauzeti puno memorije! Pazite na:

  • Duljinu sample-a
  • Sample rate (niži = manje memorije)
  • Format (Mono = manje memorije)

Optimizacija memorije

  • Koristite Mono umjesto Stereo kada je moguće
  • Smanjite Sample Rate ako kvaliteta dozvoljava
  • Obrezite nepotrebne dijelove
  • Brišite neiskorištene sample-e

Koristan savjeti

Kvalitetna snimka

Za najbolju kvalitetu snimite u tihoj prostoriji, koristite kvalitetan mikrofon, i podesite input level pravilno.

Backup

Vaše sample-e možete spremiti na USB. Organizirajte ih u mape i napravite backup.
